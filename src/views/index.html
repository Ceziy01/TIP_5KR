Каталог рецептов

API сервер для управления рецептами

Доступные эндпоинты API:

GET /api/recipes

Получить все рецепты

GET http://localhost:3000/api/recipes

GET /api/recipes/:id

Получить рецепт по ID

GET http://localhost:3000/api/recipes/1

GET /api/recipes/search

Поиск рецептов

GET http://localhost:3000/api/recipes/search?q=борщ

Фильтрация:

  • ?category=супы
  • ?difficulty=легкая
  • ?maxTime=30

POST /api/recipes

Создать новый рецепт

POST http://localhost:3000/api/recipes 
{
    "title": "Название рецепта",
    "category": "категория",
    "ingredients": ["ингредиент1", "ингредиент2"],
    "instructions": "Инструкция приготовления",
    "cookingTime": 45,
    "difficulty": "средняя"
}

PUT /api/recipes/:id

Обновить рецепт

PUT http://localhost:3000/api/recipes/1

DELETE /api/recipes/:id

Удалить рецепт

DELETE http://localhost:3000/api/recipes/1
